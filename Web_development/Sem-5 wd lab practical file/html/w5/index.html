Input Devices
An input device is any hardware device that sends data to a computer, allowing you to interact with and control it.
Output Devices
An output device is any hardware device used to send data from a computer to another device or user.
Processing Devices
Processing devices are the components responsible for the processing of information within the computer system.
Storage devices
A storage device is an integral part of the computer hardware that stores information/data to process the result of any computational work.
